Minimally configured dotfiles.
<br>
<br>

**Dep**
- [vim-plug](https://github.com/junegunn/vim-plug)

**For**
- bash
- nvim

**Do**

```sh
$ ln -s ~/.config/nvim/init.vim $REPO/init.vim
$ ln -s ~/.config/nvim/plugins.vim $REPO/plugins.vim
$ ln -s ~/.bash_profile $REPO/bash_profile
```
